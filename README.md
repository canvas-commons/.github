# Canvas Commons

A community-driven fork of [Motion Canvas](https://github.com/motion-canvas/motion-canvas),
along with some additional repos for the overall Motion Canvas ecosystem.

## Why Fork?

- Motion Canvas hasn't had updates in a while, and current maintainers don't
  have push access to the repo, nor release permissions.
- There are some features in Motion Canvas that are not yet implemented, as well
  as a couple small bugs that have been reported and not yet fixed. We want to
  fix them and add some new features.
- Ultimately, Motion Canvas is aarthificial's vision, and ours might be
  different than that. So this serves as our own version of the project.

## Important Repos

- [canvas-commons][canvas-commons] - The main
  repo for Canvas Commons.
- [awesome-canvas-commons][awesome] - A collection of Motion Canvas / Canvas
  Commons ecosystem resources.
- [examples][examples] - A collection of examples.
